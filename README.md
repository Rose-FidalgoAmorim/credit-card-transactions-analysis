# Credit Card Transactions Analysis

Exploratory data analysis of 50,000 real credit card transactions using Python.

## Objective
Analyse consumer spending behaviour across categories, demographics and time patterns to uncover key insights relevant to the fintech industry.

## Key Insights
- Grocery purchases are the most frequent (after the gas transport) and the highest in average amount ($118), suggesting bulk buying habits
- Gender has no significant impact on spending patterns, though men show slightly higher median transaction values
- Profession has a significant impact on the spending habits : the top 3 has strongly expenditures compared to the others in the top 10.
- Transaction volumes peak at 10 PM (2,600 transactions/hour) but average amounts are highest between 10 PM and 11 AM ($74–$83)
- Monday and Sunday are the busiest days; Wednesday is the quietest (mid-week low)
- Daytime transactions tend to be frequent but small : late-night transactions are rarer but higher in value

## Tools
- Python, Pandas, Matplotlib, Seaborn

## Dataset
[Credit Card Transactions Dataset — Kaggle](https://www.kaggle.com/datasets/priyamchoksi/credit-card-transactions-dataset)
1.85M transactions — sample of 50,000 rows used for analysis.
